# Deriving-Labels-for-Clinical-Risk-Prediction-Models
Brain tumors are abnormal cell growths in the brain, either benign or malignant. Early detection improves treatment outcomes. MRI scans aid diagnosis, but manual analysis is slow and error-prone. Machine Learning, especially CNNs, automates detection with higher accuracy

Brain tumors are one of the most serious and life-threatening neurological disorders caused by abnormal and uncontrolled growth of cells in the brain. These tumors can be classified into two main types: benign (non-cancerous) and malignant (cancerous). Malignant tumors are more dangerous as they grow rapidly and can spread to other parts of the brain, affecting normal brain functions. Early detection of brain tumors is extremely important because it increases the chances of successful treatment and improves patient survival rates. Magnetic Resonance Imaging (MRI) is commonly used in medical diagnosis for brain tumor detection because it provides high-resolution images of soft brain tissues, allowing doctors to clearly observe abnormalities.

However, manual analysis of MRI scans is a challenging task as it requires high expertise, is time-consuming, and may lead to human errors due to fatigue or misinterpretation. To overcome these limitations, Artificial Intelligence (AI) and Deep Learning techniques are widely used to automate medical image analysis. In this project, a Convolutional Neural Network (CNN) is implemented for brain tumor detection and classification from MRI images.

CNN is a deep learning architecture specifically designed for image processing. It works by automatically extracting important features from images such as edges, textures, patterns, and shapes without requiring manual feature engineering. The CNN model consists of multiple convolutional layers that apply filters to extract features from MRI images. These layers are followed by max-pooling layers, which reduce the spatial dimensions of the image while preserving important information, making the model more efficient and reducing overfitting.

After feature extraction, the output is flattened into a one-dimensional vector and passed through fully connected dense layers, which perform high-level reasoning and classification. A dropout layer is also used to prevent overfitting by randomly disabling neurons during training. The final output layer uses Softmax activation, which provides probability values for each class, allowing the model to classify images into glioma tumor, meningioma tumor, pituitary tumor, or no tumor.

The model is trained using a labeled MRI dataset consisting of multiple tumor classes. The dataset is divided into training, validation, and testing sets. During training, the model learns patterns from MRI images by minimizing the loss function using the Adam optimizer. The loss function used is categorical crossentropy, which is suitable for multi-class classification problems. The training process helps the model improve its accuracy by adjusting weights through backpropagation over multiple epochs.

Model performance is evaluated using different performance parameters such as test accuracy, loss, confusion matrix, and classification report. Test accuracy measures the overall correctness of predictions, while loss indicates prediction error. The confusion matrix provides a detailed view of correct and incorrect predictions for each class, and the classification report gives precision, recall, and F1-score for deeper analysis of model performance.

Finally, the trained model is used for real-time prediction of MRI images. It predicts the tumor type, assigns a tumor stage, and provides a confidence score indicating the reliability of the prediction. This makes the system useful as a decision-support tool in medical diagnosis, helping doctors in early and accurate detection of brain tumors.
